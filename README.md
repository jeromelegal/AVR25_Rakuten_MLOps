Rakuten MLOps
==============================

## How to use
### Build the services
```bash
docker-compose up --build
```

### Start services
```bash
docker-compose up --force-recreate
```
The `--force-recreate` option is required for vault to be able to initialize


# Data storage
## Data Lake
The images, models and results are stored in S3-like buckets using Minio as the server.
The buckets generated by default are:
- `raw-images`
- `processed-images`
- `models`
- `ci-results`
- `results`
- `train-images`
- `test-images`
An API is also provided to create, list or delete buckets.
